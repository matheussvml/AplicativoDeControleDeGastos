# 🚀 Inicio Rápido

## 1️⃣ Instalar dependências

```bash
npm install
```

## 2️⃣ Iniciar servidor de desenvolvimento

```bash
npm run dev
```

Abra http://localhost:5173/ no seu navegador

## 3️⃣ Para produção

```bash
npm run build
```

## ✨ Primeira vez usando?

1. **Configure seus cartões**
   - Clique no botão ⚙️ no canto inferior direito
   - Edite o nome e limite de cada cartão
   - Clique em "Salvar Alterações"

2. **Adicione seu primeiro gasto**
   - Clique no botão + (flutuante no canto inferior direito)
   - Preencha os campos
   - Se quiser parcelar, escolha "Parcelas"
   - Clique em "Adicionar Gasto"

3. **Visualize seus gastos**
   - Dashboard: Veja resumo de cada cartão
   - Gastos: Lista completa com filtros
   - Estatísticas: Gráficos por categoria

4. **Navegue entre meses**
   - Use os botões de "Anterior" e "Próximo" para mudar de mês
   - Clique em "Hoje" para voltar ao mês atual

## 💡 Dicas

- **Parcelamento**: Se você gasta R$ 300 em 3x, o app mostrará R$ 100 em cada mês
- **Filtros**: Combine múltiplos filtros para encontrar o que procura
- **Deletar**: Ao deletar um gasto parcelado, escolha entre deletar uma ou todas as parcelas
- **Alertas**: Quando atingir 80% do limite, o cartão ficará amarelo. Acima de 100%, ficará vermelho

## 📱 Responsivo

O app funciona perfeitamente em:

- ✅ Celular
- ✅ Tablet
- ✅ Desktop

Todos os dados são salvos automaticamente no localStorage do seu navegador. Não se perdem ao recarregar a página!
